# demo-audit-diffs-using-custom-z-utility-wheel-ansible-module-oef
This repository contains the sources to build a custom z/OS utility packed into a wheel driven by a custom Ansible module deployed to private automation hub and driven by OEF Git and curl


To run this demo with the shell scripts, in this order:
- `./setup.sh`
- `./build-wheel.sh`
- `./test-wheel.sh`
